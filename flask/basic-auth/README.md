# Flask Web Framework
HTTP Basic Authentication.

## Local Installation
```
virtualenv env
source env/bin/activate
pip install -r requirements.txt
```

## Start The Server

### Example 1
Barebones Flask example without using any extensions.

```python example1.py```

### Example 2
Using Flask-BasicAuth extension.

```python example2.py```
